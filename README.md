# labs

## Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/noxtgm/labs/main/boot.sh)
```

## Shell commands

- Show files: `ls`
- Show files (including hidden): `lsa`
- Show files with extra info: `lsx`
- Show files (including hidden) with extra info: `lsxa`

- Show tree on 2 levels deep: `tree`
- Show tree (including hidden) on 2 levels deep: `treea`
- Show tree on 2 levels deep with extra info: `treex`
- Show tree (including hidden) on 2 levels deep with extra info: `treexa`
