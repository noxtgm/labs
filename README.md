# labs

## Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/noxtgm/labs/main/boot.sh)
```

## Shell commands

- Show files grouped by folders and with last modification date: `ls`
- Show files (including hidden) grouped by folders and with last modification date: `lsa`
- Show files grouped by folders with file size, permisions, and last modification date: `lsx`
- Show tree on 2 levels deep listing git status for each file: `tree`
- Show tree (including hidden) on 2 levels deep listing git status for each file: `treea`
